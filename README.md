# clock.nvim

> Time to stand reminders

## Install

```lua
{ "Asthestarsfalll/clock.nvim" }
```

Optionally, install [nvim-notify](https://github.com/rcarriga/nvim-notify) for more prominent notifications.

## Usage

```lua
require("clock").setup()
```

## Options

```lua
require("clock").setup({
    default_inter = 1,
    default_info = ticktack! ticktack! ticktack!",
    icon = "‚è∞",
})
```
